1. Enter your player name
2. Select "Join (connect to game)"
3. Enter host IP address (e.g., 127.0.0.1 for local, or peer's IP)
4. Enter host port number (e.g., 5000)
5. Enter your local port (e.g., 5001)
6. Select your Pokemon
7. Battle!

This project was developed with assistance from AI as a learning tool for network programming concepts. All core logic, protocol implementation, and battle mechanics were designed and implemented through collaborative problem-solving.
